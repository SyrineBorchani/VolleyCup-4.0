## 📋 About the Project

VolleyCup 4.0 is the official front-end website for a university volleyball tournament organized at ENSI (National School of Computer Science), Tunisia. The site gives students and university teams everything they need — from discovering the event and viewing the schedule to registering their team online. It is designed to be visually bold, fast, and fully static with no dependencies or frameworks required.

## ⚙️ Tech Stack

- **HTML5** — semantic, accessible markup
- **CSS3** — custom properties, flexbox, grid, keyframe animations
- **No frameworks. No dependencies. No build tools.**

## 🚀 Getting Started

No installation or build step needed. Just clone the repo and open in your browser:

```bash
git clone https://github.com/your-username/volleycup.git
cd volleycup
open home.html
```

Or simply double-click `home.html` to open it locally.
